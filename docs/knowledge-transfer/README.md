# Knowledge Transfer Hub

These guides onboard each kind of person who interacts with WealthWise. Each one answers the same five questions, tailored to that audience:

1. **What is WealthWise?**
2. **Why does it exist?**
3. **How does this person use it?**
4. **What responsibilities do they have?**
5. **What value do they receive?**

Pick your guide:

| Guide | For |
|---|---|
| [Founders](founders.md) | Founders, leadership, investors who must explain and steer the product |
| [Developers](developers.md) | Engineers building the platform |
| [Designers](designers.md) | UX/UI designers shaping the experience |
| [Support](support.md) | Support (and compliance-adjacent) staff |
| [Advisors](advisors.md) | Financial professionals serving clients |
| [Agency Owners](agency-owners.md) | Firms running teams of advisors |
| [Clients](clients.md) | End users managing their money |

All guides draw on the shared documents: [Vision](../vision.md), [Modules](../modules.md), [Business Rules](../business-rules.md), [Principles](../principles.md), and the [WealthWise Bible](../wealthwise-bible.md).

> These are *orientation* documents — enough to understand WealthWise from a given seat. They are deliberately free of code and technology.
