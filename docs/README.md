# WealthWise — Phase 0: Product Discovery & Knowledge Base

> **The single source of truth for what WealthWise is, why it exists, and who it serves — before any code is written.**

Welcome to the WealthWise documentation hub. This `/docs` library is the **business foundation** of the product. It is intentionally free of code, APIs, schemas, and technology choices. Its only job is to create a **shared understanding** of the product across founders, developers, designers, advisors, agencies, compliance, investors, and end users.

If you are new to WealthWise, start with the [WealthWise Bible](wealthwise-bible.md) — it is the master reference. Everything else in this folder expands on a section of the Bible.

---

## What is WealthWise (in one sentence)?

WealthWise is an **AI-powered financial ecosystem** that helps people manage money independently, with AI guidance, or with verified human advisors and wealth-management agencies — always giving the user the choice.

It is **not** just a budgeting app. The long-term ambition is to become **the operating system for personal finance**.

## Documentation rules (Phase 0)

1. **No code.** No APIs, schemas, or technology choices live here.
2. **Business first.** Every document explains *why*, *who*, and *value* before *what*.
3. **One source of truth.** When two documents disagree, the [WealthWise Bible](wealthwise-bible.md) wins.
4. **Living documents.** These files evolve. Each future phase updates the relevant sections rather than starting new ones.
5. **Plain language.** Anyone — technical or not — should be able to read any page.
